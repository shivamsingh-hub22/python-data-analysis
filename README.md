# python-data-analysis
Data analysis on COVID dataset using Python (Pandas &amp; Matplotlib)

## Description
This project involves data analysis on a real-world COVID dataset using Python. The dataset was loaded and explored using Pandas, and key insights were extracted through data analysis techniques.

## Tools Used
- Python
- Pandas
- Matplotlib

## Work Done
- Loaded CSV dataset using Pandas
- Explored data using head(), tail(), and describe()
- Performed data cleaning and basic analysis
- Created visualizations using Matplotlib

## Outcome
The project helped in understanding data trends and improving data analysis and visualization skills using Python.
